N Lab.について

N Lab.はフリーランスのWebプログラマとして活動しているわたなべの個人事業（屋号）です。
Web制作・iOSアプリ開発を得意としています。詳しくは「業務内容」をご覧ください。

屋号の由来

1つのことだけを事業としてずっとやっていくより、その時その時、興味のあるもの、喜ばれるもの、時代に合ったものを生業とするため屋号は様々な事業を行っても矛盾がおきないもの、また新しい何かを始めたり続けるための実験場とする意味合いも込めて、N（名前の頭文字）+　Lab.（Laboratry/実験室）としました。

iOSアプリ開発

Objective-Cを用いたiOSアプリ開発を承っております。サーバー通信が必要なアプリにも対応。工程は基本設計から実装・テスト、ご希望によりAppleへの代行申請・リジェクト対応までデザインを除いてほぼ全て承っております。また、アプリのサーバー側だけや一部機能のみ実装など部分対応も可能です。

Objective-C PHP MySQL

Web制作（Webサイト・CMS・Webアプリケーション）・システム構築

コーディング・プログラミングのみ承っております。静的サイトやランディングページのコーディング、WordPressのテーマ制作・カスタマイズ、Jsでのフロント実装、DB設計、サーバーサイド実装、プロトタイプ制作など幅広く対応しております。

HTML CSS JavaScript jQuery Meteor.js PHP WordPress WordPress（Genesis） Java MySQL SQLServer VB.NET FuelPHP YiiFramework ZendFramework
